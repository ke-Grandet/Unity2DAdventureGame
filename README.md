## 简单说明
* 个人项目
* 是一个2D跳跃过关游戏
* 共有两关
* 第一关有奖励物品，拾取可获得分数；接触地刺、移动电锯时游戏失败；利用下落平台、弹簧、上行风扇等机关到达右边的出口
* 第二关有移动的敌人，踩在头上将其击败；小心下落的石头怪；触碰奖杯获得游戏胜利
